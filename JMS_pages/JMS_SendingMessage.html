Send message to a queue using JMS:

In this section we will see how to send a mesage to a queue hosted on ActiveMQ server.


Tools Required:

You will need the below tools in order to send a message using JMS

  • Docker: to host activeMQ container
  • IntelliJ: to code and configure JMS

    • Setup Required

    1. Install Docker on your machine.
    2. Create and run ActiveMQ on docker.
    3. Create a maven project on IntelliJ and add the following JMS and ActiveMQ dependencies.
    4. Install the jar files under Files->Project Structure->Modules->Dependencies tab. Click on + and add the jar files.

    Steps to create a message producer using JMS API:

    1. Create a connectionFactory object using the JMS classic API.
    2. Create a connection using the connectionFactory object created in the previous step.
    3. Create a session object from connection.
    4. Create a queue using createQueue method on the ActiveMQ server.
    5. Now using session create a messageProducer.
    6. Create the message you want to send from five message types JMS provides. For our example we have use TextMessage.
    7. Send the message using messageProducer send method.

    Please refer to the below code:

            
          ConnectionFactory connectionFactory = new ActiveMQConnectionFactory(ActiveMQConnectionFactory.DEFAULT_BROKER_URL);
          Connection connection = connectionFactory.createConnection();
          Session session = connection.createSession(false, Session.AUTO_ACKNOWLEDGE);
          Destination destination = session.createQueue("testJMS");
          MessageProducer messageProducer = session.createProducer(destination);
          TextMessage textMessage = session.createTextMessage("This message needs to be sent to queue");
          messageProducer.send(textMessage);
